# PlanA Assignment

UI Tests - > Cypress
API Tests - > pactumjs

## Getting Started

* Prerequisites 
```
Make sure below mentioned are installed

[nodejs](https://nodejs.org/en/)
[pnpm](https://pnpm.io/installation)
```

* On first clone, execute below command

Run commands on root folder
```
pnpm install
```
* To run UI tests 
```
pnpm cypress open
```
* To run API Tests
```
pnpm api-tests
```
